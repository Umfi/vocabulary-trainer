## Vocabulary Trainer

This is a hybrid app build with Ionic Vue. 

- Group vocables in boxes
- Import vocables from excel
- Multiple modes (typing, multiple-choice, letters)
- Practice mode using SM2 algorithm
- Translated (English and German)
